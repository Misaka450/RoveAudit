# 运营商数据门户平台（Data Portal）产品需求与技术方案

## 1. 项目概述

### 1.1 项目背景

建设一个面向运营商内部使用的数据门户平台。

当前业务数据已存储于 Doris 数据库。

平台主要用于：

* 数据清单展示
* 数据清单下载
* 数据可视化分析
* 异常数据展示
* 权限控制管理
* 后续新增清单快速接入

平台定位：

> 运营商内部统一数据门户平台。

---

## 1.2 建设目标

实现：

### 数据统一展示

所有业务清单统一入口访问。

### 权限统一管理

不同人员看到不同内容。

### 数据统一分析

统一图表分析能力。

### 配置化扩展

新增清单无需修改前端代码。

# 6. 配置化清单设计

目标：

新增清单无需开发。

---

## 配置表

sys_report

字段：

```text
id

report_name

report_code

category

description

sql_content

enable_download

enable_chart

status
```

---

## 工作流程

新增清单：

1. Doris创建数据表
2. 配置SQL
3. 分配权限
4. 自动出现在菜单

无需重新发布系统。
